
# Sorry Project (updated)
Enhanced big pink heart with 3D folding animation + touch/click interaction that displays "I MISS U".

## Build
mvn clean package

## Jenkins
- Create Jenkins credentials (username/password) with id `tomcat-creds` (Tomcat manager user).
- Set `TOMCAT_URL` in the Jenkins job (e.g. http://tomcat-host:8080).
